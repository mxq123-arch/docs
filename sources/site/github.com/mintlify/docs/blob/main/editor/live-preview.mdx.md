Preview your site while you edit. Live previews render pages exactly as they look when published, and update as you make changes to content, navigation, and configuration.

## Open live preview

Click the live preview button in the top bar to open a live preview in a new tab.

[![The live preview button in the toolbar.](https://github.com/mintlify/docs/raw/main/images/editor/live-preview-light.png)](https://github.com/mintlify/docs/blob/main/images/editor/live-preview-light.png)

[![The live preview button in the toolbar.](https://github.com/mintlify/docs/raw/main/images/editor/live-preview-dark.png)](https://github.com/mintlify/docs/blob/main/images/editor/live-preview-dark.png)

Live preview is only available for public sites.

## Live preview versus preview deployments

| | Live preview | Preview deployment |
| --- | --- | --- |
| **Speed** | Instant | Requires a build |
| **Access** | Local URL | Shareable URL |
| **Use when** | Reviewing changes while editing | Sharing with your team for review |
| **Availability** | While the editor is open | As long as the branch exists |

Use live preview for immediate feedback while editing. When you need to share changes with your team or test on a different device, use a [preview deployment](https://github.com/mintlify/docs/blob/main/editor/branching-and-publishing#preview-your-changes).
