\## 询问我们的文档

使用键盘快捷键 Command + I（macOS）或 Ctrl + I（Windows/Linux）启动与基于我们文档训练的 AI 助手的聊天。

\## 观看视频教程

前往我们的 [YouTube](https://www.youtube.com/@GetMintlify/videos) 频道，查看关于使用 Mintlify 的教程和指南。

\## 消息支持

从你的[控制台](https://dashboard.mintlify.com/)向我们发送消息。在侧边栏中选择 **Support**。

[![在控制台中打开的联系支持面板。](https://github.com/mintlify/docs/raw/main/zh/images/support-panel-light.png)](https://github.com/mintlify/docs/blob/main/zh/images/support-panel-light.png)

[![在控制台中打开的联系支持面板。](https://github.com/mintlify/docs/raw/main/zh/images/support-panel-dark.png)](https://github.com/mintlify/docs/blob/main/zh/images/support-panel-dark.png)

我们的目标是在 24 小时内回复所有消息，但在繁忙时段可能会有所延迟。

\## 通过电子邮件联系

如果你无法访问控制台，请发送邮件至 [support@mintlify.com](mailto:support@mintlify.com)。请在邮件中附上你的[控制台](https://dashboard.mintlify.com/) URL，包含你的组织和部署信息，以便我们更快地为你提供帮助。例如，`dashboard.mintlify.com/your-org/your-deployment`。
