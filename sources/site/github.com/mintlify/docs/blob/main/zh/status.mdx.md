Mintlify 维护了一个公开的状态页面，您可以在此查看其服务的实时运行状态，并查看历史可用性数据。

\## 访问状态页面

前往 [status.mintlify.com](https://status.mintlify.com) 查看 Mintlify 服务的当前状态。

状态页面展示以下服务的运行情况：

- **Documentation** - 托管的文档站点
- **Updates** - 文档更新与部署服务
- **Dashboard** - Mintlify 控制台与管理界面

\## 订阅更新

订阅后，当我们检测到事件、解决事件或安排计划维护时，你将收到通知。

1. 前往 [status.mintlify.com](https://status.mintlify.com)。
2. 点击 **Subscribe to updates**。
3. 选择你的通知方式（电子邮件、RSS，或通过 Slack 接收 RSS）。

\## 报告问题

如果你遇到的问题未在状态页显示，请通过 [support@mintlify.com](mailto:support@mintlify.com) 联系 Mintlify 支持团队。
