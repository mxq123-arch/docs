## Get started in 4 steps

<Steps>
  <Step title="Install the CLI">
    Install the Mintlify CLI to preview your docs locally and run quality checks.

    ```bash
    npm i -g mint
    ```
  </Step>
  <Step title="Connect your repository">
    Link your GitHub or GitLab repository in the Mintlify dashboard to enable automatic deployments on every push.
  </Step>
  <Step title="Write your first page">
    Create an MDX file, add frontmatter with a `title` and `description`, and write your content using Markdown and Mintlify components.
  </Step>
  <Step title="Publish">
    Push your changes or click **Publish** in the web editor. Your documentation site updates in seconds.
  </Step>
</Steps>
