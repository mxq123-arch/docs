The Mintlify agent is an AI tool that creates pull requests with proposed changes to your documentation based on your prompts. You can interact with it from the dashboard, your Slack workspace, or the API. Rather than replacing your review process, the agent works within it — every suggested change goes through a pull request so your team stays in control.

## What the agent does

When you send the agent a request, it works through a structured process before opening a PR:

- **Researches**: Searches and reads your existing documentation, any connected repositories, relevant context, and the web.
- **Plans**: Creates a structured task list for complex documentation work.
- **Writes**: Generates or updates content following writing standards and best practices.
- **Validates**: Runs Mintlify CLI checks to ensure the documentation builds correctly.
- **Creates a PR**: Opens a pull request with proposed changes for your team to review.

<Note>
  All changes go through pull requests. The agent never commits directly to your main branch.
</Note>

## Connect your GitHub account

By default, the agent opens pull requests attributed to the Mintlify bot. To attribute pull requests to your own GitHub account instead, connect your GitHub account on the [My profile](https://app.mintlify.com/settings/account) page of the dashboard.

## Connect repositories as context

The agent can only access repositories that you connect through the Mintlify GitHub App. This gives you precise control over what source code and content the agent can read when researching and writing. Configure which repositories the agent can access in the agent panel **Settings** or directly in the [GitHub App settings](https://github.com/apps/mintlify/installations/new).

## Use AI tools alongside the agent

The agent works asynchronously through pull requests, but you can also use AI coding tools like Cursor or Claude Code locally for fast, iterative edits. Install the Mintlify [skill](/ai/skillmd) and connect the [MCP server](/ai/model-context-protocol) so your local editor has the same context the agent uses — keeping all your documentation work consistent.

To let an external agent edit your content directly through MCP tool calls, connect it to the [Mintlify MCP server](/ai/mintlify-mcp).
